# ENTSO-E Energy Charts Pro

Professional WordPress plugin for energy market analysis with interactive charts powered by real-time ENTSO-E Transparency Platform data.

## Features

- Interactive charts (Line, Bar, Pie) with Chart.js
- Real-time ENTSO-E API integration
- Preconfigured chart templates
- Multi-country comparisons
- Elementor drag & drop widgets
- Mobile responsive design
- 30+ European countries supported

## Installation

1. Upload plugin ZIP to WordPress
2. Activate in Plugins menu
3. Get free API key from transparency.entsoe.eu

## Usage

1. Edit page with Elementor
2. Add ENTSO-E Chart widgets
3. Choose from preset templates or customize
4. Charts load real-time energy data automatically

## Supported Data

- Day-ahead electricity prices
- Actual power consumption
- Generation by energy source
- Cross-border electricity flows

## Requirements

- WordPress 5.0+
- PHP 7.4+  
- Elementor (free or pro)
- Free ENTSO-E API key
